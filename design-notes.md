# Análisis de Diseño - Fundación Quiero Trabajo

## Colores Principales

- **Coral/Naranja**: #EA6A5A (color principal del logo y botones)
- **Gris oscuro**: #5A6670 (texto del logo "FUNDACIÓN QUIERO TRABAJO")
- **Beige/Crema**: #F5F1E8 (fondo de secciones)
- **Blanco**: #FFFFFF (fondo principal)
- **Negro**: #000000 (texto)

## Tipografía

- Fuente moderna y limpia
- Sans-serif
- Peso variado (regular, medium, bold)

## Estilo Visual

- **Minimalista y profesional**
- **Espacios amplios** (mucho white space)
- **Imágenes grandes** y emotivas
- **Botones redondeados** con el color coral
- **Tarjetas con sombras suaves**
- **Diseño responsive**

## Elementos Clave

- Logo con el corazón/persona coral
- Botón "Haz tu donación" prominente en coral
- Fotografías de alta calidad de las beneficiarias
- Texto en español, claro y directo
- Enfoque en el impacto social

## Aplicación al Sistema de Reservas

La aplicación web usará:
- Color coral (#EA6A5A) para botones y elementos interactivos
- Fondo beige/crema para secciones
- Tipografía sans-serif limpia
- Diseño minimalista con espacios amplios
- Tarjetas para mostrar los slots disponibles
- Botones redondeados
